# Epidemic

#### 介绍
疫情人数统计系统

#### 软件架构
软件架构说明

#### 使用说明

1.  整合了SpringBoot、SpringMVC、Mybatis、SpringSecurity框架
2.  实现了用户权限分配和身份验证
3.  普通用户可以查看各省份每日新增患病人数、各省份患病总人数、治愈人数
4.  管理员可以修改删除添加各省份每日患病、总人数、控制首页折线图数据
